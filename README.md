## What I'm building

**[homecore](https://github.com/lpedro-s/homecore)** — self-hosted
infrastructure on Debian/Docker: Traefik, Pi-hole + Unbound, Grafana,
Vaultwarden, Gitea, Jellyfin, and more. Automated healthchecks, drift
detection, DNS-over-TLS, fail2ban. Built to understand systems, not
just to use them.

**[ai-jail](https://github.com/lpedro-s/ai-jail)** — sandboxing tool
in Rust for isolating AI process execution. Blocks unauthorized
filesystem access, prevents data exfiltration, restricts network calls
to what's strictly necessary. Inspired by thinking seriously about what
it means to run untrusted code.
